Deskripsi Objek UI - FR-01 Membuat atau Masuk Akun

Tabel Deskripsi Objek UI

ID. LAYAR NAMA LAYAR DESKRIPSI
P001 Page Registrasi User Page untuk membuat akun baru sebagai User/Mahasiswa

P004 Page Login Admin Page untuk masuk ke akun Admin

Antarmuka P001: Page REGISTRASI User

Id_Objek JENIS LABEL Keterangan
Title1 Title/Header Create an Account Header halaman registrasi
Subtitle1 Subtitle Enter your details to register Subjudul penjelasan halaman registrasi
TabButton1 Tab Button User Jika diklik, akan menampilkan form registrasi User
TabButton2 Tab Button Admin Jika diklik, akan menampilkan form registrasi Admin
Textfield1 Inputfield/Textfield Full Name Space kosong untuk input nama lengkap pengguna
Textfield2 Inputfield/Textfield Username Space kosong untuk input username pengguna
Textfield3 Inputfield/Textfield Password Space kosong untuk input password pengguna
Textfield4 Inputfield/Textfield Confirm Password Space kosong untuk konfirmasi password pengguna
Button1 Button Sign Up Jika diklik, akan menyimpan biodata dari user dan membuat akun baru
Link1 Hyperlink Sign in Jika diklik, akan memindahkan user ke halaman login
