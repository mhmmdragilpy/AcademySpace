1. Pendahuluan

1.1 Tujuan

Dokumen User Acceptance Test (UAT) ini disusun untuk memvalidasi bahwa sistem Academy Space telah memenuhi kebutuhan fungsional sesuai dengan implementasi nyata. Pengujian dilakukan oleh end-user untuk memastikan sistem siap digunakan dalam lingkungan produksi.

1.2 Ruang Lingkup

  • Fungsi Autentikasi (Sign In, Sign Up, Reset Password, Logout)
  • Manajemen Profil Pengguna
  • Pencarian dan Pemesanan Fasilitas
  • Manajemen Reservasi (User)
  • Panel Administrasi (Admin)
  • Manajemen Fasilitas (Admin)
  • Analytics dan System Tokens (Admin)

1.3 Referensi Implementasi

Teknologi: Next.js 14, TypeScript, NextAuth, PostgreSQL

Button Labels: Sign In, Sign Up, Check Availability, Approve, Reject

Navigation: Home, My Reservations, Admin Panel, Guide, Profile
