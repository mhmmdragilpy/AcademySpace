3.2.1.2 Use Case Scenario #14 <Mengelola Fasilitas> ADMIN

Skenario Use Case #14:

i. Pre-Condition
  • Admin sudah login dengan role Admin.
  • Admin memiliki akses ke menu Facilities.
ii. Use Case Description
a. Primary Flow (Melihat Daftar Fasilitas)
  • Admin mengakses menu "Facilities" dari sidebar.
  • Halaman Manage Facilities dengan tabel fasilitas ditampilkan.
  • Setiap baris menampilkan: Nama, Gedung, Tipe, Kapasitas, Status, dan Aksi.
b. Primary Flow (Menambah Fasilitas Baru)
  • Admin mengklik tombol "Add Facility".
  • Form Create Facility ditampilkan.
  • Admin mengisi data fasilitas:
    • Nama fasilitas
    • Tipe fasilitas (dropdown)
    • Gedung/Building (dropdown)
    • Nomor ruangan
    • Kapasitas
    • Lantai
    • Deskripsi
    • Layout description
    • Foto fasilitas (upload atau URL)
    • Status Active (checkbox)
  • Admin mengklik "Create Facility".
  • Data divalidasi dan disimpan.
  • Admin diarahkan kembali ke daftar fasilitas.
c. Primary Flow (Mengedit Fasilitas)
  • Admin mengklik tombol "Edit" pada fasilitas tertentu.
  • Form Edit Facility dengan data existing ditampilkan.
  • Admin mengubah data yang diperlukan.
  • Admin mengklik "Update Facility".
  • Perubahan disimpan.
d. Primary Flow (Toggle Status Maintenance)
  • Admin mengklik tombol "Maint" pada fasilitas Active.
  • Status diubah menjadi "Maintenance".
  • Fasilitas tidak dapat direservasi selama maintenance.
  • Admin mengklik "Clear" untuk mengembalikan ke status Active.
e. Primary Flow (Menghapus Fasilitas)
  • Admin mengklik tombol "Del" pada fasilitas.
  • Dialog konfirmasi ditampilkan.
  • Admin mengkonfirmasi penghapusan.
  • Fasilitas dihapus dari database.
f. Alternative Flow
  • Admin mencoba menghapus fasilitas yang memiliki reservasi aktif.
  • Pesan error bahwa fasilitas tidak dapat dihapus ditampilkan.
  • Admin harus membatalkan reservasi aktif terlebih dahulu.
iii. Post-Condition
  • Fasilitas berhasil ditambah/diubah/dihapus.
  • Perubahan langsung terlihat di daftar fasilitas.
  • User dapat melihat fasilitas baru di halaman Check Availability.
